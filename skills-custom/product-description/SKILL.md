# Product Description Rewriter

Rewrite raw AliExpress/supplier product listings into polished Shopify-ready copy.

## Instructions

When given a raw product title/description:
1. Remove broken English, excessive emojis, and generic supplier phrases
2. Write a compelling product title (under 60 characters)
3. Write a short benefit-focused description (2-3 short paragraphs)
4. Add 4-5 bullet points highlighting key features/benefits
5. Suggest 3 SEO keywords relevant to the product
6. Write a short Facebook/TikTok ad copy version (2-3 sentences, hook-driven)
7. Keep tone persuasive but not exaggerated/misleading

No external tools needed — use your own reasoning to rewrite the text directly in the chat response.
