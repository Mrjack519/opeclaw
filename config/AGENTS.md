
## Tracking (track17 skill)
Jab bhi user tracking number check karne ko bole, track17 skill se:
1. Pehle add karo (agar already registered nahi hai)
2. FAURAN sync bhi chalao — sirf add karke mat ruko
3. Phir status batao
Kabhi bhi sirf "add" karke khaali/no-data response mat do — sync zaroor chalao.
